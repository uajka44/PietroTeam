# Piętro Cafe — Przepisy

Aplikacja mobilna z przepisami dla pracowników Piętro Cafe.

## Adres po wdrożeniu
https://uajka44.github.io/pietro-przepisy/

## Struktura projektu
```
pietro-przepisy/
├── index.html        ← cała aplikacja (jeden plik)
└── images/           ← zdjęcia kaw (dodać później)
```

## Jak aktualizować przepisy
Otwórz `index.html`, znajdź sekcję `DANE — PRZEPISY` (szukaj `const DRINKS`).
Każda kawa to obiekt z polami: `name`, `category`, `icon`, `tags`, `variants`.

## Jak wdrożyć zmiany przez Fork
1. Otwórz Fork → repozytorium `pietro-przepisy`
2. Stage changes → Commit → Push
3. Zmiany są live na GitHub Pages po ~30 sekundach
